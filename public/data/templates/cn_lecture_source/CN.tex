\begin{SummaryBox}
本讲介绍常微分方程的基本形式、平衡点及其几何意义，并通过相图理解系统的整体行为。
\end{SummaryBox}

\section{常微分方程的基本形式}

考虑系统：
\[
\dot{x} = f(x)
\]

其中 $x(t)\in \R^d$，函数 $f$ 描述系统的演化规律。

\begin{definition}[平衡点]
若 $f(x_0)=0$，则称 $x_0$ 为系统的平衡点。
\end{definition}

\begin{AnalysisBox}
平衡点表示系统“静止”的位置。一旦进入该点，系统将不再变化。
\end{AnalysisBox}

---

\section{相轨线与相图}

解 $x(t)$ 在状态空间中的轨迹称为相轨线。

\begin{ExampleBox}
考虑一维系统：
\[
\dot{x} = x(1-x)
\]

其平衡点为 $x=0$ 和 $x=1$。
\end{ExampleBox}

\begin{AnalysisBox}
通过符号分析可以判断：
\begin{itemize}
\item $x=0$ 为不稳定点
\item $x=1$ 为稳定点
\end{itemize}
\end{AnalysisBox}

---

\section{线性系统}

考虑线性系统：
\[
\dot{x} = Ax
\]

\begin{theorem}
系统的稳定性由矩阵 $A$ 的特征值决定。
\end{theorem}

\begin{AnalysisBox}
\begin{itemize}
\item 实部 < 0：渐近稳定
\item 实部 > 0：不稳定
\item 有正有负：鞍点
\end{itemize}
\end{AnalysisBox}

---

\section{一页总结}

\begin{itemize}
  \item 常微分方程描述动态系统
  \item 平衡点是关键分析对象
  \item 相图提供整体行为的直观理解
  \item 线性系统是非线性系统分析的基础
\end{itemize}
